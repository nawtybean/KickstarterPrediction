Kickstarter Api

The goal behind this API is for other developers to use this prediction model in their own applications.
The API end point for predicting Kickstarter Campaigns is: https://nawtybean3d.pythonanywhere.com/kickstarter/kickstarter/

Headers: application/json
Type: POST
Auth: None

Sample Json:

"backers": "350",
"campaign": "30",
"goal": "50000",
"category": "3",
"country": "1",
"staff_pick": "on"}
